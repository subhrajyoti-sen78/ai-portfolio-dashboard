# AI Portfolio Dashboard

Welcome to the AI Portfolio Dashboard! This is a modern, sleek, and responsive portfolio designed to showcase your AI-generated capabilities, AI models, and predictive analytics projects. It features a stunning "dashboard-like" UI as requested, mimicking a high-end SaaS platform for a premium feel.

## Features

- **Dashboard Layout:** A professional sidebar navigation with a search bar and categorized links.
- **Data Visualization:** CSS-only animated bar charts that showcase "AI Model Performance" dynamically without heavy JavaScript libraries.
- **KPI Tracking:** Callout cards for your best performing models, tracking tokens/sec, latency, or other metrics important to your clients.
- **Responsive Design:** Fluid layout that adapts cleanly to different screen sizes.
- **Aesthetic Excellence:** Custom color palettes featuring deep navy, bright blue, and energetic greens with smooth box shadows for a "tacky yet beautiful" premium aesthetic.

## Deployment via Netlify

Because this is a completely static, vanilla HTML/CSS project, it is extremely fast and easy to deploy to Netlify.

### How to deploy in 30 seconds (No Git or CLI required)

1. Open your web browser and go to [Netlify Drop](https://app.netlify.com/drop).
2. Log in or sign up if you haven't already.
3. Open your file explorer and locate the folder containing this website (`c:\Users\User\Downloads\New website`).
4. Drag and drop the entire `New website` folder into the Netlify Drop area on the screen.
5. Wait a few seconds, and Netlify will provide you with a live, shareable link to your new portfolio!

*(Note: Your `.env` file was found empty, and Git/Node.js were not detected on your Windows environment, which is why the automated GitHub push and Netlify CLI deployment could not be completed. The drag-and-drop method above is the fastest alternative!)*
